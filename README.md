# ChatGPT--chat.iyunci.cn
# **介绍**

基于openai的GPT-3.5 turbo （ChatGPT）API接口实现的数字方块AI，可实现与官网ChatGPT接近的持续性对话效果。同时支持自定义prompt的方式，让**数字方块AI**成为你工作中的伙伴，成为私人工作助理或者企业员工助理。

# **强调**已支持私有化部署
